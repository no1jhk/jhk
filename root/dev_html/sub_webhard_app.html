작업내용 LG U+ 웹하드 모바일앱 동영상스트리밍 서비스
작업기간 2015.04.04 - 2015.04.12
고객사 LG U+ 웹하드
URL http://www.webhard.co.kr/

기존 LGU+ 웹하드 모바일앱에 동영상스트리밍서비스를 추가하는 작업을 진행하였습니다.
동영상 서비스를 리스트보기, 섬네일 보기로 구분하여 시청할 수 있도록 구성하였고, 여러 동영상 확장자에 맞춰 아이콘도 작업하였습니다.
보안상 이유로 원본PSD 없이 가이드만으로 진행하게 되었고, 사이즈는 720x1280로 작업하였습니다.
