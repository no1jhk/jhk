와이즈포스 음식점 포스기기 내부 UI 구축작업

작업내용 WISEPOS 포스 기기 UI 작업
작업기간 2013.07.23 ~ 2013.08.07.
고객사 WISEPOS

모바일, 태블릿이 아닌 생소한 결제수단으로 사용하는 포스기기의 UI를 작업을 진행하였습니다. 기존의 단순 기능에만 초점을 둔 UI에서 통일성 있고 무게감 있는 디자인을 적용하여 가독성 및 사용성 향상에 중점을 두고 작업하였습니다. 플랫한 느낌을 유지하면서도 그라디언트를 적절히 사용해 볼륨감 있는 버튼이 되도록 작업하였습니다. 컬러 배색은 카페베네의 컬러를 참조하여 작업하였습니다.


  • 갤럭시탭 기준으로 작업되었고 가변형으로 적용되는 기기에 알맞게 적용되도록 하였다.
